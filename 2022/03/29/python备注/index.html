备注

函数中【*】和【**】区别

*: 表示接收的参数作为元组来处理

**: 表示接收的参数作为字典来处理

如果 *和**同时在一个函数参数中使用时，*必须在**之前，否则出错

_的作用

-有4种使用场景

  1. 交互式解释器中保存最后一个表达式的值
  2. 指向被忽略的值
  3. 字面上分割数字
  4. 赋予属性和方法特性含义

__name__ == __main__: 表示在当前文件中,此判断可以避免测试代码在模块被导入后执行

__file__: 表示当前python脚本的目录地址

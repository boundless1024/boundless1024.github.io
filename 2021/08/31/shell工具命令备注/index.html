shell cut、sed记录

发表于2021-08-31|更新于2022-03-07|shell
cut命令
参数 说明
-b 以字节为单位进行分割
-c 以字符为单位进行分割
-d 自定义分隔符，默认为制表符
-f 提取的列号，需要与-d一起使用

  • who

  • 例如 who | cut -d o -f 1

    sed
参数 说明
-e 以选项中指定的script来处理输入的文本文件
-f 以选项中指定的script文件来处理输入的文本文件
-h 显示帮助
-n 仅显示script处理后的结果
-V 显示版本信息
a 新增， a 的后面可以接字串
c 取代， c 的后面可以接字串
d 删除
i 插入， i 的后面可以接字串
p 打印，亦即将某个选择的数据印出
s 取代 s/old/new//g

  • 例如 adb shell pm path com.etoury.ui

  • 例如2 adb shell pm path com.etoury.ui | sed ‘s/package://g’

  • 例如 sed ‘4a\ 0——-= ‘ sed-demo.txt

    快捷键
— —
Ctrl + L 清除屏幕并将当前行移到页面顶部。
Ctrl + A 跳到光标所在行的行首。
Ctrl + B 向行首移动一个字符，与左箭头作用相同。
Ctrl + C 中止当前正在执行的命令。
Ctrl + D 删除光标位置的字符,没有任何字符-关闭 Shell 会话。
Ctrl + E 跳到光标所在行的行尾。
Ctrl + F 向行尾移动一个字符，与右箭头作用相同。
Ctrl + K 从光标位置删除到行尾。
Ctrl + T 光标位置的字符与它前面一位的字符交换位置。
Ctrl + U 从光标位置删除到行首。
Ctrl + W 删除光标前的单个域。
